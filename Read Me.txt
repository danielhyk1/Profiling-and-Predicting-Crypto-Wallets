Predicting and profiling cryptocurrency wallets (ETH) based on their transactional behaviors.

Predicts volatility of the wallets using GARCH.
Predicts transactional value using ARIMA and EMA.
Categorizes using k means clustering.

find_addresses.py: 
	Web Scraping on blockchain APIs
histogram.py:
	Creates graphs and charts based on address data
k_means.py:
	Unsupervised ML using k means on various attributes of crypto wallets
models.R:
	GARCH, ARIMA, and EMA models conducted on transaction history
synthesis.py:
	Combines data from crypto addresses, models, and k means into csv files
transaction_filtering.py:
	Filters the addresses found and outputs the data into csv files.
